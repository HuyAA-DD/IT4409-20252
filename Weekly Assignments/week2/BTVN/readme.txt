Bài 1: Trang thông tin lịch học (TKB.html)
+ Trang cấu trúc của sematic html cơ bản và đáp ứng các yêu cầu về header và footer, không CSS hay JS
+ thẻ <header> và <head> chứa thông tin trang
+ trong thẻ <main> sẽ chứa 2 section chính : Thông tin sinh viên và Bảng thời khóa biểu
+ Thông tin sinh viên được tạo bởi thẻ <img> và <list>
+ Bảng thời khóa biểu được tạo vởi thẻ <table> 
+ Ảnh cá nhân được lưu trong file image

Bài 2: Trang Google Search (simple_gg.html)
+ Trang cấu trúc của sematic html cơ bản và đáp ứng các yêu cầu về header và footer, không CSS hay JS
