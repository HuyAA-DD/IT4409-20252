# Chức năng phân trang trong React (Pagination)

## 1. Mục tiêu
Hiển thị danh sách dữ liệu theo từng trang thay vì hiển thị toàn bộ một lần, giúp:
- Giao diện gọn gàng hơn
- Tăng hiệu năng
- Dễ thao tác với dữ liệu lớn

---

## 2. Các state cần thiết

```js
const [currentPage, setCurrentPage] = React.useState(1);
const [itemsPerPage, setItemsPerPage] = React.useState(5);
```

- currentPage: trang hiện tại
- itemsPerPage: số dòng mỗi trang

---

## 3. Tính số trang

```js
const totalPages = Math.ceil(filteredUsers.length / itemsPerPage);
```

---

## 4. Cắt dữ liệu (quan trọng nhất)

```js
const startIndex = (currentPage - 1) * itemsPerPage;
const endIndex = startIndex + itemsPerPage;

const paginatedUsers = filteredUsers.slice(startIndex, endIndex);
```

---

## 5. Hiển thị dữ liệu

```jsx
{paginatedUsers.map((u) => (
    <tr key={u.id}>
        <td>{u.name}</td>
    </tr>
))}
```

---

## 6. Điều hướng trang

```js
function goToPrevPage() {
    if (currentPage > 1) {
        setCurrentPage(currentPage - 1);
    }
}

function goToNextPage() {
    if (currentPage < totalPages) {
        setCurrentPage(currentPage + 1);
    }
}
```

---

## 7. UI phân trang

```jsx
<button onClick={goToPrevPage}>Trang trước</button>

<span>Trang {currentPage} / {totalPages}</span>

<button onClick={goToNextPage}>Trang sau</button>
```

---

## 8. Chọn số dòng mỗi trang

```jsx
<select
    value={itemsPerPage}
    onChange={(e) => setItemsPerPage(Number(e.target.value))}
>
    <option value={5}>5</option>
    <option value={10}>10</option>
    <option value={20}>20</option>
</select>
```

---

## 9. Reset trang khi thay đổi

```js
React.useEffect(() => {
    setCurrentPage(1);
}, [keyword, itemsPerPage]);
```

---

## 10. Công thức quan trọng

```js
startIndex = (currentPage - 1) * itemsPerPage
endIndex = startIndex + itemsPerPage
```

---

## 11. Tổng kết

Luồng hoạt động:
1. Lọc dữ liệu (filter)
2. Tính tổng số trang
3. Cắt dữ liệu (slice)
4. Hiển thị
5. Điều hướng bằng setCurrentPage
